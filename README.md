# QQ 群课表追踪

AstrBot 插件：通过用户上传的 `.ics` 日历文件绑定 QQ 群友课表，支持查询当前是否在上课、本周课表，以及每日课时统计图片。

## 指令

* `绑定课表`：绑定发送者 10 分钟内上传/转发的 `.ics` 文件。
* `如何添加课表`：查看绑定课表的说明文本，回复内容可在 WebUI 配置。
* `在上课吗 @某人`：查询群友当前上课状态。
* `看看课表 @某人`：查看群友本周课表。
* `课表隐私 公开`：公开当前状态和完整课表。
* `课表隐私 状态可查`：只允许查询当前状态。
* `课表隐私 私密`：禁止他人查询，并不进入日报。
* `课表日报 开启`：开启本群每日课时统计日报，仅本群课表管理员可用。
* `课表日报 关闭`：关闭本群每日课时统计日报，仅本群课表管理员可用。
* `课表日报 状态`：查看本群日报开关状态。
* `课表撤回 状态`：查看本群 ICS 文件和查询图片自动撤回状态。
* `课表撤回 文件 开启/关闭/跟随配置`：设置本群上传 `.ics` 消息自动撤回，仅本群课表管理员可用。
* `课表撤回 图片 开启/关闭/跟随配置`：设置本群“在上课吗/看看课表”图片自动撤回，仅本群课表管理员可用。
* `删除课表`：删除自己的课表绑定。

## 配置

在 AstrBot 插件配置中可调整：

* `timezone`：默认 `Asia/Shanghai`
* `daily_report_time`：默认 `22:30`
* `how_to_add_schedule_reply`：`如何添加课表` 命令的回复文本
* `daily_report_enabled_groups`：在 WebUI 中预先开启日报的群号列表
* `group_admins`：在 WebUI 中配置群课表管理员，填写 JSON 文本，格式为“群号 -> 管理员 QQ 号列表”
* `auto_recall_ics_uploads`：是否默认自动撤回群成员上传/转发的 `.ics` 文件消息
* `auto_recall_schedule_images`：是否默认自动撤回“在上课吗/看看课表”生成的图片
* `schedule_image_recall_seconds`：查询图片撤回延迟，默认 `90`

群内 `课表撤回` 指令会覆盖 WebUI 默认值；使用 `跟随配置` 可恢复跟随 WebUI。

## 依赖

安装插件目录下的 `requirements.txt`：

```bash
pip install -r requirements.txt
```
