# QQ 群课表追踪

AstrBot 插件：通过用户上传的 `.ics` 日历文件绑定 QQ 群友课表，支持查询当前是否在上课、本周课表，以及每日课时统计图片。

## 指令

* `绑定课表`：绑定发送者 10 分钟内上传/转发的 `.ics` 文件。
* `在上课吗 @某人`：查询群友当前上课状态。
* `看看课表 @某人`：查看群友本周课表。
* `课表隐私 公开`：公开当前状态和完整课表。
* `课表隐私 状态可查`：只允许查询当前状态。
* `课表隐私 私密`：禁止他人查询，并不进入日报。
* `删除课表`：删除自己的课表绑定。

## 配置

在 AstrBot 插件配置中可调整：

* `timezone`：默认 `Asia/Shanghai`
* `daily_report_time`：默认 `22:30`

## 依赖

安装插件目录下的 `requirements.txt`：

```bash
pip install -r requirements.txt
```
